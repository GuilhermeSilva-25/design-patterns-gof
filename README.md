# Design Patterns (GoF) - Técnicas de Programação II

Repositório destinado aos exercícios da disciplina ministrada pelo Professor **Vinicius Heltai Pacheco**. 
O objetivo é implementar e compreender os padrões de projeto do GoF utilizando diferentes linguagens.

## 🚀 Tecnologias Utilizadas
* **Java**
* **Python**
* **JavaScript**

## 📂 Organização do Repositório
Os padrões estão divididos em três categorias principais:
1. **Criacionais**: Foco em como os objetos são criados.
2. **Estruturais**: Foco em como classes e objetos são compostos.
3. **Comportamentais**: Foco na comunicação entre objetos.
